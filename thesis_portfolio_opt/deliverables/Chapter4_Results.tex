\section{Headline Result}
The end-to-end integrated system---combining autonomous hyperparameter search (AutoResearch, 72 experiments), multi-agent swarm intelligence (MiroFish, 14 agents), and macro-based ML portfolio optimization---achieves an out-of-sample Sharpe ratio of \textbf{1.330--1.410} (2022--2024) depending on transaction cost assumptions, returning +19.6--20.7\% annualized vs SPY's +10.0\%, with lower drawdown ($-$17.8--18.1\% vs $-$24.5\%).

\section{Key Findings}
\begin{enumerate}
\item \textbf{Integrated ML-Only achieves Sharpe 1.330--1.410} (2.3--2.5$\times$ SPY), using AutoResearch-optimized LightGBM with maxW=0.6
\item \textbf{Swarm overlay trades return for safety}: ML+Swarm Sharpe=0.860 but improves drawdown by 1.5\% and reduces volatility by 2.9\%
\item \textbf{SVR discovered by feedback loop}: AutoResearch identified SVR as untried; it scored \#2 (Sharpe 0.868, highest IC=0.200)
\item \textbf{Concentration wins}: maxW=0.6 $>$ 0.5 $>$ 0.4 $>$ 0.35 $>$ 0.3 monotonically
\item \textbf{PCA hurts}: experiments with PCA dimensionality reduction scored near zero; macro features are already informative
\item Top predictive features: 10Y Treasury yield, VIX, 2Y yield, BBB corporate spread
\item Transaction cost break-even: $\sim$25 bps (viable at institutional costs 5--15 bps)
\item OOS Sharpe difference is \textbf{not} statistically significant ($p = 0.79$) due to short 3-year sample
\item Fama-French 5-factor $R^2 = 0.38$; 62\% of returns unexplained by standard factors
\item Walk-forward RandomForest (Phase 2) achieves Sharpe 0.969 with lowest drawdown ($-$12.3\%)
\item \textbf{Regime-conditional hurts}: VIX-based model switching (4 variants) underperforms; splitting reduces training data too aggressively
\item \textbf{IC-weighted ensemble middling}: LGBM+SVR+Ridge ensemble (Sharpe 0.656) trails single LGBM by 30\%
\item 72 experiments across 4 rounds of autonomous search, 5 feedback loop iterations
\end{enumerate}

\section{MiroFish Swarm Intelligence Impact}
The MiroFish multi-agent system (14 agents, 35 simulation rounds) provides a ``risk overlay'' rather than an alpha signal:
\begin{itemize}
\item Agent agreement (mean 0.247) is used as a risk scaling factor in the CVXPY optimizer
\item When agents disagree, position sizes are reduced; when they agree, full allocation
\item Net effect: $-$0.470 Sharpe delta but $+$0.016 drawdown improvement and $-$0.029 volatility reduction
\item Verdict: \textbf{beneficial\_risk} --- useful for risk-averse investors, not for return maximization
\end{itemize}

\section{Statistical Significance}
Ledoit-Wolf test for Sharpe ratio equality (OOS only): $z = 0.27$, $p = 0.79$. The 3-year OOS window is too short to establish statistical significance at conventional levels. This is an honest limitation that motivates longer evaluation periods in future work.

\section{Closed-Loop Architecture}
The system operates as a closed feedback loop:
\begin{enumerate}
\item \textbf{AutoResearch} (57 experiments) identifies optimal model configuration
\item \textbf{MiroFish} (14 agents) generates risk overlay and swarm features
\item \textbf{Integrated Pipeline} runs walk-forward backtest with both inputs
\item \textbf{Feedback Loop} analyzes results and generates suggestions for the next AutoResearch iteration
\end{enumerate}
Five iterations completed, each improving on the previous (standalone Sharpe: 0.803 $\to$ 0.832 $\to$ 0.897 $\to$ 0.938; integrated: 1.330 at tc=10bps, 1.410 at tc=3bps).
